- 🚀 **Incoming SDE Intern @ Amazon** — joining the Search team in Palo Alto, Summer 2026
- 🥗 **Co-Founder @ [Supatrack](https://supatrack.app)** — AI-powered nutrition tracking, live on iOS & Android with a 4.9+ rating

- 🌐 **System Architect** — past work on India's ONDC government commerce grid, microservices at scale
- 📚 **Currently learning** — distributed systems internals, CLRS, and how Amazon Search actually works at scale
